-tooloption: v6eval tool option

INITIALIZATION

  • Reboot NUT (reboot.rmt)
  • Configure site-local address (manualaddrconf.rmt)
  • Enable HA function (mip6EnableHA.rmt)
  • Add default router into routing table of NUT (route.rmt)
