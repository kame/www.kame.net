Y2K issues with KAME code

KAME only supplies updated networking code in operating system kernel and userland. Therefore, KAME code should introduce no new Y2K issues. You may want to check Y2K information for the original operating system distributions, like:
  • KAME/FreeBSD: http://www.freebsd.org/
  • KAME/NetBSD: http://www.netbsd.org/
  • KAME/BSDI: http://www.bsdi.com/
  • KAME/OpenBSD: http://www.openbsd.org/
If you find any Y2K problem caused by KAME code, please report via our normal bug reporting path. If you find any Y2K problem in original operating system distributions, the bug report should go to each of the operating system distribution sources, listed above.
