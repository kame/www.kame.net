Submit a kame problem report


Thank you for taking the time to let us know about a problem with KAME. Please fill out the form as completely as possible. Make sure you fill in the "Environment" field as requested with the output from the machine on which problem occurred.

Please be sure to EXPLICITLY STATE the following items:

  • The operating system you are using (for example, NetBSD)
  • The version of operating system you are using (1.4 for example)
  • CPU architecture you are using (sparc for example)
  • The KAME kit releasing tree (SNAP, STABLE or RELEASE)
  • KAME kit release date (19991231)
  • If you are reporting for userland application on non-KAME IPv6 stack, the detailed version/platform/whatever info about the stack.
  • Complete information to reproduce the problem. For example, exact command line, generated results (captured by script(1)), routing setups (netstat -rn), whatever necessary.
NO JAPANESE TEXT IN BUG REPORT PLEASE! The text will be get archived, and will be sent to English-only Mailing list.
