KAME IPv6-patched applications

We KAME group make many patchkits for making popular applications IPv6-ready. Our collection includes famous mozilla, apache, cvs, python, perl, ucd-snmp, and other random applications, you-name-it.

Application patches can be found at ftp://ftp.kame.net/pub/kame/misc/ and its mirrors.

Accessing cvs repository

We work on these patches on temporary working area in our cvs repository. If you are in hurry, or you would like to keep track of those changes easily, you may want to use anonymous CVS, rather than the finished patch kit. Please, however, notice the following facts:
  • Code in the repository is an unfinished work.
    Please do not route questions about the content of the repository, if you are unsure about the status. When reporting bugs, please check finished patchkit in ftp://ftp.kame.net/pub/kame/misc/. If we do not issue patchkit, that means we do not have a working version for that.
  • Content on the repository are not supposed to work in many cases.
    For example, in case we imported third-party applications into repository but had no time to tackle, the repository will not do much help.
  • Sometimes they are obsolete.
    We try to submit these patchkit to the original author as possible, so that it will be merged into the main distribution. We do not maintain patchkit after the merge, and in this case content of repository will be obsolete (and left as is),
  • They may not be based on the most recent version.
    We try to base our patchkit on the most recent "released" version of a third-party software. Because of this, we do not usually try to hack alpha versions.
  • Please be sure to join snap-users mailing list
    For announcements and updates. Click here for details.

Okay, enough said. Here's how you can access our cvs repository for application patches: 

% cvs -d :pserver:anoncvs@cvs.kame.net:/cvsroot/apps login
(enter "anoncvs" to "password:" prompt)
% cvs -d :pserver:anoncvs@cvs.kame.net:/cvsroot/apps co -P apache13
For checking module name (like apache13 in the above), please use cvsweb source code browser.

getting commit notifications

You can get commit notification emails for KAME application repository, by contacting majordomo@kame.net, with the following email content:
  • subscribe cvs-apps
